PROJECT TITLE: Connect 4 Game
PURPOSE OF PROJECT: Computer game "Connect 4 Game"
VERSION or DATE: 14.07.2015
HOW TO START THIS PROJECT: Run java Main
AUTHORS: Eugene Guzik
USER INSTRUCTIONS:
After the game starts, use "Start a New Game" button to start a new game.
Click "Play Next Move" button to do the player's step.
Click "Play to End" button to do the quick game to the end.
In Main class you can change the strength of the players (RandomAgent, BeginnerAgent, IntermediateAgent, AdvancedAgent, BrilliantAgent or MyAgent).
